# Blink Personal

Your **private AI-powered app builder**. Turn ideas into full Next.js applications instantly.

Built as a personal alternative to Blink.new with full control over your data and tokens.

## ✨ Features

- AI Agent with persistent memory
- Real-time preview
- Multi-file editing (Next.js + TypeScript + shadcn/ui)
- Image generation with Flux
- Local SQLite + Supabase support
- One-click export to GitHub + Vercel
- Dark mode + responsive sidebar

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/blink-personal.git
cd blink-personal

# 2. Install dependencies
npm install

# 3. Run development server
npm run dev
